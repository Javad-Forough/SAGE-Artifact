# Output Layout

Generated outputs are written here by the wrapper scripts.

- `RESULTS/smoke/`: outputs from the reduced verification run
- `RESULTS/main/`: main evaluation CSVs
- `RESULTS/appendix/`: appendix-specific CSVs
- `RESULTS/figures/`: regenerated figure files
- `RESULTS/work/`: temporary databases, key files, TPM state, and other
  generated run state
- `RESULTS/logs/`: optional script logs

This directory is safe to delete and regenerate.
