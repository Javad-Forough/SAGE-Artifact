# Requirements

## Base Requirements

- Linux or another recent POSIX-like environment
- Python 3.10 or newer
- a writable local filesystem

The artifact was prepared against Python 3.12 on Linux x86-64.

Install Python dependencies with:

```bash
pip install -r requirements.txt
```

## Optional Requirements for Specific Results

### Agent Correctness Table

Required only for `SCRIPTS/run_agent_eval.sh`:

- local Ollama installation and running daemon;
- a local model available to Ollama, by default `llama3.2`.

The artifact does not contact any external hosted model API.

### TPM Overhead Table

Required only for `SCRIPTS/run_tpm_eval.sh`:

- `tpm2-tools`
- either:
  - a real TPM or firmware TPM accessible to `tpm2-tools`, or
  - `swtpm` for the documented simulation path

## No External Data

- No external datasets are required.
- All workloads are synthetic and generated by the experiment code.
- No cloud KMS account, remote database, or proprietary data is needed.
